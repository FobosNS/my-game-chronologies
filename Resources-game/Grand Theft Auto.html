Клод Спид 2D // GTA 2

Единственный герой 2D-эры с четкой предысторией. Работает на банды в Anywhere City. Был заморожен в крио-камере (согласно фильму-тизеру).

Герои GTA 1 2D // 1997

Бубба, Кивуан, Кэти, Микки, Сонечка, Трэвис, Трой, Ульрика. Восемь преступников на выбор игрока.

Виктор Вэнс 3D // VC Stories

Капрал армии, втянутый в наркоторговлю. Пытался спасти больную семью, но стал главой империи Вайс-Сити.

Томми Версетти 3D // Vice City

«Мясник Харвуда». Предан семьей Форелли. Первый герой, захвативший город и начавший говорить.

Карл Джонсон 3D // San Andreas

Вернулся домой в гетто, чтобы найти убийц матери. Стал менеджером рэперов, пилотом и грабителем казино.

Тони Сиприани 3D // LC Stories

Преданный солдат семьи Леоне. Вернулся из изгнания, чтобы навести порядок в Либерти-Сити.

Майк 3D // GTA Advance

Мелкий преступник, ищущий убийцу своего напарника Винни в Либерти-Сити.

Клод 3D // GTA III

Молчаливый наемник. Был предан подругой Каталиной при ограблении банка. Его девиз — полное отсутствие сантиментов.

Нико Беллик HD // GTA IV

Бывший солдат из Восточной Европы. Приехал к брату Роману за «американской мечтой», убегая от призраков войны.

Джонни Клейбитц HD // Lost & Damned

Вице-президент байкерского клуба «The Lost». Пытался сохранить клуб вопреки безумию своего лидера Билли Грея.

Луис Лопес HD // Gay Tony

Правая рука короля ночных клубов Тони Принса. Балансирует между семейными проблемами и гламуром.

Хуан Ли HD // Chinatown Wars

Избалованный сын босса Триад. Должен доставить фамильный меч дяде Кенни, но попадает в засаду.

Майкл де Санта HD // GTA V

Грабитель банков в отставке под программой защиты свидетелей. Страдает от кризиса среднего возраста.

Франклин Клинтон HD // GTA V

Амбициозный парень из гетто, ищущий реальный способ выбраться из уличной грязи и заработать миллионы.

Тревор Филипс HD // GTA V

Бывший военный пилот и абсолютный психопат. Живет в пустыне, торгует оружием и не терпит оскорблений.

Люсия и Джейсон HD // GTA VI

Криминальный дуэт в современном штате Леонида. Их связывают не только преступления, но и доверие (или его отсутствие).

RenderWare — Использовался в III, VC, SA. Легко модифицируемый движок эпохи PS2.
RAGE (Rockstar Advanced Game Engine) — Мощь IV, V и будущей VI части. Отвечает за графику и мир.
Euphoria — Процедурная анимация, благодаря которой Нико Беллик так натурально шатается, будучи пьяным.
