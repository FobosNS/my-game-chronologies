⚠️ РАСШИРЕННАЯ БАЗА УГРОЗ (БЕСТИАРИЙ)

ВЕЛОЦИРАПТОР

ОПАСНОСТЬ: ВЫСОКАЯ. Самый распространенный враг. Обладает коллективным разумом. Тактика: Отстрел конечностей или использование усиленных транквилизаторов.

ТЕРИЗИНОЗАВР

СТАТУС: ТАНК. Слепой мутант с 70-сантиметровыми когтями. Реагирует на бег и выстрелы. Тактика: Медленное перемещение шагом. В бою использовать гранатомет.

ПТЕРАНОДОН

ОПАСНОСТЬ: СРЕДНЯЯ. Атакуют с воздуха. Могут поднять Регину в воздух и сбросить с высоты. Тактика: Эффективны автоматические пистолеты и быстрые перекаты.

ТИРАННОЗАВР (T-REX)

СТАТУС: ОБЪЕКТ ПРЕСЛЕДОВАНИЯ. Главный символ серии. Примечание: Стандартный арсенал бесполезен. Использовать окружение: газовые баллоны, краны, турели.

ГИГАНОТОЗАВР

СТАТУС: ТИТАН. Достигает 21 метра в длину. Сметает всё на своем пути. Примечание: Уязвим только для Спутника «Третьей Энергии» (Satellite Laser).

АЛЛОЗАВР

ОПАСНОСТЬ: ВЫСОКАЯ. Быстрее и агрессивнее рапторов. Способен проглатывать цели целиком. Тактика: Ослепляющие гранаты и огонь из тяжелого пулемета.

МОЗАЗАВР

СТАТУС: ВОДНАЯ УГРОЗА. Обитает в затопленных реакторных отсеках. Тактика: Использование подводных лаунчеров и минных ловушек.

ПЛЕСИОЗАВР

СТАТУС: МОРСКОЙ БОСС. Огромная рептилия с длинной шеей. Атакует из воды. Тактика: Прицельный огонь по шее при попытке атаки с берега.

ИНЬЯНЦИЯ (INYANCE)

ОПАСНОСТЬ: НИЗКАЯ. Мелкие падальщики-симбионты. Тактика: Не тратьте патроны, просто отпихивайте ногами при приближении.

CEBALRAI (МУТАНТ)

СТАТУС: БИО-ОШИБКА. Двуглавый гибрид Giganotosaurus. Регенерирует за секунды. Тактика: Уничтожение обоих ядер управления одновременно.

АУСТРАЛОР (AUSTRALOR)

ОПАСНОСТЬ: ЭКСТРЕМАЛЬНАЯ. Космический аналог раптора. Умеет выпускать электрические разряды. Тактика: Энергетическое оружие и укрытия.

МИКРОРАПТОР

ОПАСНОСТЬ: НИЗКАЯ. Роятся сотнями в вентиляции. Тактика: Огнемет или автоматическая турель.

ТРИНИТИ (TRINITY)

СТАТУС: ВЫСШАЯ ФОРМА. Разумный динозавр, способный управлять другими особями. Примечание: Результат секретных генетических опытов в виртуальной реальности.

КАРНОТАВР

ОПАСНОСТЬ: ВЫСОКАЯ. "Бык" среди ящеров. Атакует тараном. Тактика: Уворот в сторону и атака в уязвимый бок.

[ ФАЙЛ 01: РЕГИНА ]

Агент спецподразделения S.O.R.T. Мастер скрытных операций и взлома. Единственный оперативник, участвовавший в инцидентах на острове Ибис и в Эдвард-Сити. Статус: ЖИВА.

[ ФАЙЛ 02: ДИЛАН МОРТОН ]

Командир отряда T.R.A.T. Эксперт по тяжелому вооружению. Пожертвовал собой (предположительно), чтобы спасти данные о временном сдвиге. Статус: НЕИЗВЕСТНО.

[ ФАЙЛ 03: 3-Я ЭНЕРГИЯ ]

Проект доктора Кирка. Революционный источник энергии, использующий ионы из атмосферы. Побочный эффект: искривление пространства и перенос материи через миллионы лет.

[ ФАЙЛ 04: НОЕВ КОВЧЕГ ]

Секретный план по спасению человечества от вымирания путем переноса людей в далекое будущее. Привел к катастрофическому столкновению двух эпох.

[ ФАЙЛ 05: ЭДВАРД КИРК ]

Гениальный ученый-социопат. Инсценировал свою смерть, чтобы продолжить исследования. Считает динозавров лишь «досадным побочным эффектом» прогресса.

[ ФАЙЛ 06: ЛЮДИ В ШЛЕМАХ ]

Загадочные выжившие из будущего, утратившие дар речи. Защищают динозавров как своих «богов». Сражаются с фанатичной жестокостью.

[ ФАЙЛ 07: ОБЪЕКТ «ИБИС» ]

Частный остров, превращенный в секретную лабораторию. Место первого контакта. Сейчас полностью уничтожен системой самоликвидации.
